<h1>Gala Icons</h1>

<p>Gala is a lined icon set with subtle, yet still playful styling. The project began as a personal challenge to create a usable web icon set using only open source tools, and is free for both personal and commercial use under a GPLv3 license.</p>

<p><strong>Use it in your project with the Iconify Framework:</strong> https://icon-sets.iconify.design/gala/.</p>
